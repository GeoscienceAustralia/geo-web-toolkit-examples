Simple map


<div id="map" style="width:800px;height:600px"></div>
<ga-map map-element-id="map">
    <ga-map-layer   layer-name="Simple map layer name"
                    layer-url="http://basemap.nationalmap.gov/ArcGIS/services/USGSTopo/MapServer/WMSServer"
                    is-base-layer="true">
    </ga-map-layer>
    <ga-map-control map-control-name="mouseposition"></ga-map-control>
</ga-map>

The above code is running the sample below with the inclusion of jQuery, AngularJS, OpenLayers 2.13.1 and the GA-WebToolkit

  • The div with the id 'map' is the container for OpenLayers to create and present the map. This should be placed somewhere that angularjs isn't 'watching' to avoid problems with binding.
  • The 'ga-map' element needs to know where the container element is and is told by the 'map-element-id' attribute.
  • A ga-map element can have n layers as children which get rendered in order of declared html
  • ga-map-layer can be of type WMS,XYZTileCache or ArcGISCache. If layer-type is left empty, defaults will go off the Url provided.
  • is-base-layer attribute is needed to indicate to the framework if the layer being added is a base layer
  • layer-name is used as a non-unique identifier
  • layer url is the endpoint of the server providing the data, for WMS it should end in 'WMSServer'
  • The ga-map-control is an interface to access OpenLayers native controls. See documentation for the other supported controls.
